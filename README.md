# British Airways Reviews Dashboard

## Project Overview
This project presents an interactive data visualization dashboard analyzing customer reviews of British Airways. The dashboard was created using Tableau to explore passenger satisfaction across multiple service categories.

The dashboard helps analyze how customers rate different aspects of the airline including cabin staff service, entertainment, food & beverages, ground service, seat comfort, and value for money.

## Tools Used
Tableau
Data Visualization
Exploratory Data Analysis (EDA)

## Dashboard Insights
The average overall rating of British Airways is 3.9.

Cabin staff service received one of the highest ratings with an average of 3.4.

Entertainment has a relatively low rating of 1.3, suggesting an area where improvements may be needed.

Food & beverages and ground service have moderate ratings around 2.8 and 3.0.

Seat comfort and value for money ratings indicate mixed customer experiences.

## Dashboard Features
Monthly trend analysis of overall airline ratings.

Geographical visualization showing ratings by country.

Aircraft comparison showing which aircraft types receive better ratings.

Interactive filters allowing users to analyze data by:
Seat Type
Traveller Type
Aircraft
Continent
Date Range

## Dataset Information
The dataset contains customer reviews of British Airways including ratings for different services, traveller types, seat classes, aircraft types, and review dates.
