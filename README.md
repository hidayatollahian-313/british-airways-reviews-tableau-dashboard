# British Airways Reviews Dashboard

## Overview
This project is a data analysis and visualization dashboard created using Tableau. It focuses on analyzing customer reviews of British Airways to understand passenger satisfaction and service performance.

## Objectives
- Analyze customer ratings across different service categories
- Identify trends in airline performance over time
- Compare ratings based on aircraft, country, and traveller type
- Highlight areas of improvement

## Tools Used
- Tableau
- Data Visualization
- Exploratory Data Analysis (EDA)

## Key Insights
- Overall average rating is 3.9
- Cabin staff service has relatively higher ratings
- Entertainment has the lowest rating, indicating improvement needed
- Ratings vary across countries and aircraft types

## Features
- Interactive dashboard
- Filters for seat type, traveller type, aircraft, and date
- Monthly trends analysis
- Country-wise performance visualization

## Dataset
The dataset includes customer reviews, ratings, aircraft details, and travel information.
